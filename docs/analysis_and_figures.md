Companion to the full specification [data_analysis.md](data_analysis.md). Here: **commands**, **artifacts**, and **figure paths** only.

## Commands (pipeline)

```bash
.venv/bin/python -m data_analysis.ingest --output-root output
.venv/bin/python -m data_analysis.metrics --output-root output
.venv/bin/python -m data_analysis.plot --output-root output
```

Or:

```bash
.venv/bin/python -m data_analysis.pipeline --output-root output
```

Quick table:

| Artifact | What |
|----------|------|
| `processed/manifest.parquet` (or `.csv`) | All `raw/solutions/**/*.json`, flattened (pandas + pyarrow optional). |
| `processed/paired_metrics.parquet` | Manifest + BF reference columns and ratios. |
| `processed/summary_by_config.csv` | Naive aggregates (feasible rate, mean energy steps, etc.; **excludes SA**). |
| `processed/energy_curves_agg.parquet` / `.csv` | Per-step stats of `energy_history` scaled **per instance** by `abs(ref_objective_value)` — **SA excluded**. |

Generated by [data_analysis/plot.py](../src/data_analysis/plot.py) from non-empty `paired_metrics` and/or `energy_curves_agg`. Names and meaning match [data_analysis.md](data_analysis.md) (energy history and benchmark sections); summarized below.

## Figures (`output/images/` subfolders)

First plot run removes **legacy** PNGs that used to live directly under `images/` (same basenames as before).

### `images/energy_history/` (from `energy_curves_agg`)

One PNG **per QAOA depth** `p` (suffix `_p1`, `_p2`, …). **Y axis:** mean ± σ of **per-instance** \(f / |f^*_\mathrm{inst}|\) on a **single shared scale** (both formulations are already normalized).

| PNG pattern | Cohort / use |
|-------------|----------------|
| `cudaq_qubo_tvirt_n5_p{p}.png` | QUBO vs **V-QAOA** (CUDA-Q `tqudo_virtual`), `n=5`; one axis. |
| `cirq_tqudo_vs_cq_tvirt_n5_n9_p{p}.png` | **N-QAOA** vs **V-QAOA** for **`n=5` and `n=9`** (four series); shared axis. |
| `cirq_tqudo_by_n_p{p}.png` | **N-QAOA** only (Cirq `tqudo`); one series per `n_cities` at fixed \(p\). |

Dashed lines: optimum level \(\pm 1\) after the same scaling (see `data_analysis.md`). Not raw tour cost.
